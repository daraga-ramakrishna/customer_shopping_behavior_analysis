# Customer_Shopping_Behavior_Analysis
End-to-end Customer Behavior Data Analytics project using Python, SQL, and Power BI. Includes data cleaning, EDA, SQL business queries, dashboard creation, and a full analytical workflow showcasing customer insights, segmentation, KPIs, and data-driven decision-making.
Customer Behavior Data Analyst Portfolio Project

A complete end-to-end data analytics project demonstrating how raw customer data is transformed into business insights using Python, SQL, and Power BI. This project follows an industry-standard analytical workflow and highlights skills essential for Data Analyst roles.

**Project Overview**

This project analyzes customer shopping behavior to uncover patterns, segments, revenue drivers, and actionable insights. It includes:

Data Cleaning & Preparation

Exploratory Data Analysis (EDA)

SQL Business Queries

Interactive Power BI Dashboard

Insight Summary & Recommendations

**Tech Stack**

Python: Pandas, NumPy, Matplotlib, Seaborn

SQL: Joins, Aggregations, Window Functions

Power BI: Data Modeling, DAX, Interactive Visuals
